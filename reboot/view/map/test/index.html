Votre navigateur n'est pas compatible avec ODGE.

Les navigateur dont la compatibilité a été vérifiée sont:

  • Chrome 5.0+
  • Firefox 3.6+
  • Opera 10.5+
  • Safari 5.0+

Des versions plus anciennes peuvent éventuellement être compatibles. Pensez tout de même à mettre à jour votre navigateur.

La meilleur expérience de jeu est obtenue avec Chrome ou Opera.
